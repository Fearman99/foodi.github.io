# Restaurant Reservation and Online Food Ordering Website

## Overview

This repository contains the source code for a restaurant reservation and online food ordering website. The application allows users to make restaurant reservations and order food online for a seamless dining experience.

## Features

- **Restaurant Reservation:**
  - Users can browse available restaurants.
  - Book tables for a specified date and time.

- **Online Food Ordering:**
  - Browse menu items and place orders.
  - Specify delivery or pickup options.

- **User Accounts:**
  - Create accounts to manage reservations and orders.
  - View order history and reservation details.

## Tech Stack

- Frontend: css
- Backend: node js
- Database: mysql
